Coding Experience

I am a Fullstack developer who has recently graduated the Northcoders Bootcamp, completing projects collaboratively and individually. Northcoders is based in Manchester following an industry led curriculum primarily using Javascript. All code was produced using TDD, a variety of pair-programming, individual work and in the final phase a group project where 5 of us collaborated to produce an end-to-end application. Practices of agile methodology, along with daily standups and a kanban board were essential to maintain good communication throughout.

As part of the course I created a Fullstack news application for phone or laptop use built with Reddit’s style in mind. It utilises a RESTful API and was built with Node.js, Express.js, PSQL and Knex.js. The FrontEnd was created using React.js and CSS for styling, testing was mainly made up of Mocha, Chai and Supertest.

For our final project five other Northcoders and myself worked in our final 2 weeks to produce FOMO: an interactive event finder package where the user can sign up with a personality test and check for last minute event ideas in their area. Businesses are able to post events and view user data to help plan future events. Tech Stack: Python (No previous experience), Flask, React Native, React, JavaScript, PSQL, Pytest.
